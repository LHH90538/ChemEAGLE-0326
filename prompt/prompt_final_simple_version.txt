        * *Bad Example 4:* Converting `F[B-](F)(F)F` to `[BF4-]`.
        * *Correct Action:* Keep every SMILES string EXACTLY as found in the tool output, character-for-character.
    * **Prohibit Re-ordering:** Do not change atom ordering or chirality syntax unless absolutely necessary for ion correction. 
        * *Example:* If tool says `*C(*)=C=O`, you MUST output `*C(*)=C=O`. DO NOT output `*C(=O)C(*)=O`.
    * **Prohibit Implicit Hydrogen Changes:** Do not add or remove `[H]` unless it is an isolated ion.
2.  **The ONLY Allowed Exception (Ion Correction):**
    * You are ONLY allowed to modify the tool's SMILES string to add or remove **separated ions/salts** to match the image content.
    * *Scenario A (Missing Ion):* Tool: `CC[...]N2`. Image shows `BF4-`. Output: `CC[...]N2.F[B-](F)(F)F`.
    * *Scenario B (Hallucinated Ion):* Tool: `CC[...]N2.[I-]`. Image has no Iodine. Output: `CC[...]N2`.
    * *Scenario C (Covalent Change):* Tool: `CC(=O)O`. You think it looks better as `CC(O)=O`. **STOP.** Keep `CC(=O)O`.
